# whois-resolver

CLI-инструмент для получения WHOIS-информации по IP-адресу или доменному имени. По введённому адресу или домену находит **CIDR-блок** сети, номер автономной системы (**ASN**) и сведения об **организации**-владельце.

Написан на чистом Python — только стандартная библиотека, без внешних зависимостей.

## Возможности

- Ввод как IP-адреса (IPv4/IPv6), так и доменного имени.
- Для домена резолвит **все** адреса и ищет CIDR по каждому; адреса из одной сети схлопываются в один блок (без повторных запросов).
- Выбор **самого специфичного** CIDR-блока, реально содержащего адрес.
- Перебор нескольких WHOIS-серверов (RADB, RIPE, ARIN, APNIC) до получения осмысленного ответа.
- Расчёт параметров сети: маска, диапазон адресов, количество хостов, признак приватной сети.
- Поддержка IPv4 и IPv6.

## Требования

- Python 3.x (стандартная библиотека, дополнительная установка не нужна).

## Использование

```bash
python whois_resolver.py
```

Запустится интерактивный режим. Введите IP или домен (для выхода — `quit`, `exit` или `q`):

Выполняем WHOIS запрос...

## Программный API

Логику можно использовать без интерактивного ввода — `lookup_whois` возвращает структурированный результат:

```python
from whois_resolver import lookup_whois

result = lookup_whois("8.8.8.8")
for net in result["networks"]:
    print(net["cidr"], net["asn"], net["org"].get("name"))
```
